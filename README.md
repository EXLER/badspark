# badspark
BadUSB hacking toolkit for Digispark ATtiny85.

### Requirements

* Arduino IDE >= 1.6.5 ([configured for Digispark](https://digistump.com/wiki/digispark/tutorials/connecting))
* Python 3

### Usage

```bash
$ python3 badspark.py [-h] -s SCRIPT -p PORT
```
